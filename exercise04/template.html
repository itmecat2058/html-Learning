Landmark Elements!

Main Element

The main element carries the main content of the document. Informaion in the main element is directly related to information on the docuemnt or can expand upon such information


Example of Main HTML

<main>

<h3>

Main Element

</h3>

<p>

INFORMATION!

</p>

</main>

Header Element

The header element generally has things like a logo, a search form, authors name, or other elements. It also sometimes has introductory information or navigational aids.


Example of Header HTML

<header>

<h3>

Landmark Elements

</h3>

<p>

Writen by [REDACTED]

Published on [REDACTED]

</p>

</header>

Nav Element

The nav element generally has links, either to places within the same docuemnt or it links to other related documents. These can go to menus, tables, or graphs within the document.


Example of Nav HTML

<nav>

<ul>

<li><a href="#">Home</a></li>

<li><a href="#">About</a></li>

<li><a href="#">Contact</a></li>

</ul>

</nav>

Aside Element

This element is meant for informaion that isnt directly related to the informaion on the docuemnt, generally found in sidebars.


Example of Aside Elelemnt

<aside>

<p>

More information about topic

</p>

</aside>

Footer Element
